---
title: Installation
description: Install Minions Ideator packages
---

## TypeScript / Node.js

```bash
npm install @minions-ideator/sdk minions-sdk
```

## Python

```bash
pip install minions-ideator
```

## CLI

```bash
npm install -g @minions-ideator/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)

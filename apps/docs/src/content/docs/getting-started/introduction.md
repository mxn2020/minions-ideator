# Minions Ideator

Agent to help brainstorm ideas and plan features

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-ideator/sdk`) — Core library for Node.js
- **Python SDK** (`minions-ideator`) — Python bindings
- **CLI Tool** (`@minions-ideator/cli`) — Command-line interface
- **Web App** — Interactive playground at [ideator.minions.wtf](https://ideator.minions.wtf)
